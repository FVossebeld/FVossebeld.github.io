# wiki/ — private synthesis (not published)

Optional scratch space for **private** working notes and synthesis that should *not* appear
on the public site. Quartz only builds `content/`, so anything here stays internal.

Use it for:

- Half-formed synthesis that isn't ready to be a public page.
- Private cross-source analysis or planning that informs `content/` but shouldn't ship.
- Notes on `raw/` sources that aren't for publication.

When something here matures into a public page, the `wiki-ingest` skill moves a cleaned-up
version into `content/`. The public, interlinked wiki lives in `content/`; this folder is
the back room.

See [`AGENTS.md`](../AGENTS.md) for the full schema.
